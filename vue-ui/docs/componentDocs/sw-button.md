# sw-button

按钮用于开始一个即时操作。

### 示例

<br/>
<sw-button />

### 代码

```html
<sw-button />
```

### Attributes

| 属性    | 说明             | 类型                        | 是否必填 | 默认值 |
| ------- | ---------------- | --------------------------- | -------- | ------ |
| onClick | 点击按钮时的回调 | (event: MouseEvent) => void | 否       | -      |
